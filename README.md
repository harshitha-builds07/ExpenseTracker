# ExpenseTracker App

A simple Android application to track daily expenses with category-based insights and visual analytics.

## Features
- Add and manage expenses
- Category-based filtering
- Pie chart visualization
- Monthly insights dashboard

## Tech Stack
- Java
- Android Studio
- SQLite
- MPAndroidChart

## Project Structure
- `src/` → Main source code
- `res/` → Layouts and resources
- `gradle/` → Build system files
